在 MySQL 中，可以使用 ORDER BY 子句对检索到的数据进行排序。**ORDER BY** 用于根据一个或多个列对结果进行升序或降序排列。

基本语法

  1. ORDER BY 后面跟着想要排序的列名，可以是一个或多个列。
  2. [ASC|DESC] 用于指定排序方式，默认是升序 (ASC)。

在特定条件下排序

这将从居住在美国的顾客中，

  1. 先对姓氏升序；
  2. 再对于相同姓氏的顾客，按名字升序排列。

对 NULL 值排序

这将按照最后登录日期降序排序，将所有的 NULL 值放在排序结果的最后。
