Java-什么是类（Class）？

快速理解

对象表示具有状态和行为的实体，例如：一辆汽车是一个对象，汽车的颜色、价格。。

更具体的就是

  1. 状态（类的属性）：车的颜色、价格。。
  2. 行为（类的方法）：车可以前进、倒车、加速等行为；

细节说明：

  1. 类名必须和文件名相同。

构造方法的特点

  1. 名字和类名相同
  2. 不能被void修饰
  3. 可以进行重载，这就是为什么有无参构造和有参构造

类相关操作

初始化类变量有哪些方式？

  1. 初始化自定义变量

    在变量定义时直接对变量进行赋值初始化。

  2. 构造器初始化变量

    在类的构造函数（构造器）中对成员变量进行初始化。构造函数是在创建对象时调用的特殊方法。

  3. 代码块中初始化

    使用初始化块（包括静态初始化块和非静态初始化块）来对成员变量进行初始化。静态初始化块用于静态变量的初始化，非静态初始化块用于实例变量的初始化。
