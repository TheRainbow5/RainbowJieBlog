  1. 在MySQL中，当一张数据表没有与其他任何数据表形成关联关系时，可以将当前数据表直接删除。
  2. 数据和创建语句结构都被删除；
  3. 所有相关索引被删除

基本语法：

清空表

  1. TRUNCATE

    1. 删除表中所有的数据
    2. 保留创建语句结构
    3. 不持支回滚，清空后表数据无法恢复

  2. DELETE FROM

    1. 删除表中所有数据；
    2. 保留创建的语句结构
    3. 设置相关参数，则支持回滚，表数据可以恢复；

说明：

  1. TRUNCATE TABLE 在功能上与不带 WHERE 子句的 DELETE 语句相同。
